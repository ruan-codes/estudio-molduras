# Estúdio de Molduras

App web para tirar ou enviar uma foto, aplicar uma moldura e baixar/imprimir o resultado — tudo direto no navegador, sem enviar nada pra nenhum servidor.

## Funcionalidades

- Tirar foto pela câmera (frontal ou traseira) ou enviar uma imagem
- Escolher entre várias molduras prontas (temas gerais e datas comemorativas)
- Enviar sua própria moldura (PNG, WebP ou SVG com fundo transparente)
- Adicionar legenda na foto
- Baixar em PNG ou imprimir (foto única ou em tirinha de 3x/4x)

## Como rodar

```bash
npm install
npm run dev
```

Acesse o endereço que aparecer no terminal (geralmente `http://localhost:5173`).

## Build de produção

```bash
npm run build
```

## Tecnologias

React + Vite
